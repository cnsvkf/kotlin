- 변수 / 상수
    : var / val / const val / 

    - var : 변수
    - val : 상수
    - const val : main 함수 전에 선언해야 하는 형식으로 선언해야 하는 상수

        -> _형식_
        ``` kotlin
        const val MAX = 100 // 파일 최상단

        object Constants {
        const val MAX = 100
        } // 오브젝트 안

        class Test {
            companion object {
                const val TAG = "Test"
            } // 컴패니언 오브젝트 안
        }   
        ``` 
 
        - object : 하나만 쓰는 객체
        - copanion object : class 안에서 바로 object 만들어서 씀
