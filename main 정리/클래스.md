    - _toString(), 
 equals(),

    - ___copy()___ : 객체를 복사하면서 일부 값만 바꿀 수 있음
 
    - 형식(copy)
```kotlin
data class User(val name: String, val age: Int)

fun main() {
    val user1 = User("철수", 20)
    val user2 = user1.copy(age = 21)

    println(user1)
    println(user2)
} /* 결과
User(name=철수, age=20)
User(name=철수, age=21) */
```

- sealed class : 지정한 클래스만 상속받을 수 있게 하는 클래스 
    - 형식
```kotlin
sealed class Result

// 이 두 개의 클래스만 상속 가능
class Success(val data: String) : Result()
class Error(val message: String) : Result()
// 성공, 에러 클래스 정의는 다른 곳에서

fun printResult(result: Result) {
    when (result) {
        is Success -> println("성공: ${result.data}")
        is Error -> println("실패: ${result.message}")
    }
} 

fun main() {
    printResult(Success("저장 완료"))
    printResult(Error("네트워크 오류"))
}
```

- abstract class : 일부 기능은 구현하고 일부는 자식에게 강제
    - 형식
```kotlin
abstract class Animal {
    fun eat() {
        println("먹는다")
    }

    abstract fun sound()
}

class Dog : Animal() {
    override fun sound() {
        println("멍멍")
    }
}

fun main() {
    val dog = Dog()
    dog.eat()
    dog.sound()
}
```

___인터페이스는 규칙 강제, abstract은 일부 강제___

- companion object : 객체 생성 없이 클래스 이름으로 접근 가능한 영역
    - 형식
```kotlin
class MathUtil {
    companion object {
        fun sum(a: Int, b: Int): Int {
            return a + b
        }
    }
}

fun main() {
    println(MathUtil.sum(3, 4))
} // 7
```

> ## 상속 클래스 제어

- final : 상속금지(기본설정)

- open class : 상속 가능한 클래스

- override : 부모 기능 재정의
    - 형식
```kotlin
open class Parent {
    open fun greet() {
        println("안녕하세요")
    }
}

class Child : Parent() {
    override fun greet() {
        println("자식 인사")
    }
}

fun main() {
    Child().greet()
} // 결과 : 자식 인사
```

- super : 부모 기능 접근
    - 형식
```kotlin
open class Parent {
    open fun greet() {
        println("부모 인사")
    }
}

class Child : Parent() {
    override fun greet() {
        super.greet()
        println("자식 인사")
    }
}

fun main() {
    Child().greet()
} /* 결과
부모 인사
자식 인사 */
```

-> _자식 클래스에서 greet을 재정의(자식 인사) 했기에 부모 기능을 쓸려고 super_
