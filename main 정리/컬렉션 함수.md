- forEach { } : 각 요소를 하나씩 처리
    - 형식
```kotlin
fun main() {
    val list = listOf("A", "B", "C")
    list.forEach {
        println(it)
    }
} /* 결과
A
B
C */
```
- forEachIndexed { index, value -> } 
    - -> : 매개변수와 실제 실행할 코드 블록을 구분하는 역할
    - 형식
```kotlin
fun main() {
    val list = listOf("사과", "바나나", "포도")

    list.forEachIndexed { index, value ->
        println("$index : $value")
    }
} /* 결과
0 : 사과
1 : 바나나
2 : 포도 */
```
---
- withIndex() : __index와 value를 짝으로 만들어 for문에 쓸 수 있게 함__ 

    - 형식
```kotlin
fun main() {
    val list = listOf("사과", "바나나", "포도")

// index은 순서 value은 리스트 데이터값
    for ((index, value) in list.withIndex()) {
        println("$index : $value")
    }
} /* 결과
0 : 사과
1 : 바나나
2 : 포도 */

```

- filter { } : 조건을 만족하는 요소만 남김
    - 형식
```kotlin
fun main() {
    val numbers = listOf(1, 2, 3, 4, 5)
    val result = numbers.filter { it % 2 == 0 }
    println(result)
} // [2, 4]
```
- find { } / fistOrNull() : 조건 만족하는 첫요소 반환, 없으면 null
    - 형식
```kotlin
fun main() {
    val numbers = listOf(3, 7, 10, 15)
    val result = numbers.find { it > 8 }
    println(result)
} // 10
```

- any { } : or 조건으로 하나라도 있는 지, boolean 
    - 형식
```kotlin
val scores = listOf(40, 50, 90)

```

- all { } : and 조건 
    - 형식 
```kotlin

```

- sortedBy { } : 기준값으로 오름차순 정렬 
    - 형식 
```kotlin
data class User(val name: String, val age: Int)

fun main() {
    val users = listOf(
        User("철수", 25),
        User("영희", 20),
        User("민수", 23)
    )

    val result = users.sortedBy { it.age }
    println(result)
}
```

- groupBy { } : 특정 기준으로 묶어서 map으로 만듦
    - 형식
```kotlin
fun main() {
    val numbers = listOf(1, 2, 3, 4, 5)

    val result = numbers.groupBy { it % 2 }

    println(result)
} // {1=[1, 3, 5], 0=[2, 4]} -> Map 그대로 프린트 식 나오는 형식
```

- associateBy { } : 각 요소를 하나의 key로 매핑해서 map으로 만듦 + ___덮어쓰기___
    - 형식
```kotlin
fun main() {
    val numbers = listOf(1, 2, 3, 4)

    val result = numbers.associateBy { it % 2 }

    println(result)
} // {1=3, 0=4}
```
