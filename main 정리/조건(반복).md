- if(조건) : 조건문

- for (변수 in 범위) : 반복문
    ```
    range(범위) 정리

        1..10 : 1부터 10까지

        1 until 10 : 1부터 9까지

        10 downTo 1 : 10부터 1까지 감소

        1..10 step 2 : 2씩 증가
    ```

- while(조건) : 조건이 참인 동안 반복

- do { } while : 실행 후 검사
    - 형식
```kotlin
fun main() {
    var i = 1
    do {
        println(i)
        i++
    } while (i <= 3)
} /*
1
2
3 */ 
```

- __when__ : 분기에 따라 실행( =swich)
    - 형식
```kotlin
fun main() {
    val score = 2

    when (score) {
        1 -> println("아주 나쁨")
        2 -> println("나쁨")
        3 -> println("보통")
        else -> println("기타")
    }
} // 나쁨
```

->  값 반환 시 else 처리 필수
```kotlin
val result = when (x) {
    1 -> "one"
    2 -> "two"
    else -> "unknown"
} // 이유 : 모든 경우 처리해야함
```
