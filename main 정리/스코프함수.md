- let : 객체를 it로 받고, 마지막 결과 반환, null 아닐 때만 실행
    
    - 형식
```kotlin
fun main() {
    val name: String? = "철수"

    name?.let {
        println("${it}님")
    }
} // 철수님
```
---
- run : 객체 문맥 this 사용, 마지막 결과 반환

```kotlin
fun main() {
    val result = "hello".run {
        uppercase()
    }
    println(result)
} // HELLO
```
---
- apply { } :  객체 (생성/초기회/반환)을 한번에 함
    - 형식
```kotlin
data class User(var name: String = "", var age: Int = 0)

fun main() {
    val user = User().apply {
        name = "철수"
        age = 20
    }
    println(user)
} /* User(name=철수, age=20)
data class를 구현한 객체를 출력시에는 클래스 이름도 나옴 */
```
---
- also : 객체는 유지, 중간 부가 작업 수행 
    - 객체 안건들이고 작업 수행(로그)
    - 형식
```kotlin
fun main() {
    val list = mutableListOf(1, 2, 3)
        .also {
            println("현재 상태: $it")
        }

    println(list)
}
```
---
- with(객체이름) : 객체 이름 생략하고 여러 작업 처리
    - 형식
```kotlin
data class User(var name: String, var age: Int)

fun main() {
    val user = User("철수", 20)

    with(user) {
        println(name)
        println(age)
    } 
} /* 결과
철수
20 */
```
