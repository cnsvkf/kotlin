- is : 특정타입인지 검사
    - 형식
```kotlin
fun main() {
    val value: Any = "hello"

    if (value is String) {
        println(value.length)
    }
} // 결과 : 5
```
---

- isEmpty() : 길이 0인지 검사, boolean
    - 형식
```kotlin
fun main() {
    val text = ""
    println(text.isEmpty())
} // true
```
- isBlank() : 공백도 비어있다고 판단, boolean
    - 형식
```kotlin
fun main() {
    val text = "   "
    println(text.isBlank())
} // true
```

- isNotEmpty(), isNotBlank() : 반대기능

- isNullOrEmpty() : __null__ or __길이0__ 검사, boolean

- isNullOrEmpty() : __null__ or __공백__ 검사, boolean
    - ___자주 씀___
---

- isLetter() : 문자 알파벳인지 검사, boolean
    - 형식
```kotlin
fun main() {
    val ch = 'A'
    println(ch.isLetter())
} // true
```
---
- any { } : 하나라도 만족 시 true
    - 형식
```kotlin
fun main() {
    val list = listOf(1, 2, 3, 4)
    println(list.any { it > 3 })
}
```
- all { } : 전부 만족 시 true
