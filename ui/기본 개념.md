> ## UI

- view : UI 의 기본단위 
    - 버튼 하나, 텍스트 하나 = View

- ViewGroup : UI 컨테이너 역할을 하는 클래스(부모 - 자식 관계)

     -> android.view.ViewGroup
    
    - 형식

```kotlin
LinearLayout {
    Text("A")
    Button(...)
}
```

> ### Composable 함수(UI를 그리는 함수)

- __@Composable 어노테이션이 붙음__
    - 거의 무조건 ui은 붙혀야 함
- 상태가 바뀌면 자동으로 다시 실행됨 (Recomposition) 
```kotlin 
@Composable
fun A() { }
```

- Composable 안에서만 Composable 호출 가능

```kotlin
@Composable
fun A() {
    B() // 가능
}

@Composable
fun B() {
    Text("hi")
}
```
