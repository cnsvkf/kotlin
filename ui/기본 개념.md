> ## UI

- view : UI 의 기본단위 
    - 버튼 하나, 텍스트 하나 = View

- ViewGroup : UI 컨테이너 역할을 하는 클래스(부모 - 자식 관계)

     -> android.view.ViewGroup
    
    - 형식

```kotlin
LinearLayout {
    Text("A")
    Button(...)
}
```

### 레이아웃(layout) 
- viewgroup을 상속한 클래스
    - viewgroup은 view 상속
