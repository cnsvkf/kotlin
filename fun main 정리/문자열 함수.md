 #### 메타 문자(Regex 내 규칙확인) 종류(자주 쓰임)

- 위치 및 경계 지정
    - ^ : 문자열의 시작 (^Hello → Hello로 시작하는 문장)
    - \$ : 문자열의 끝을 의미 (End$ → End로 끝나는 문장)
    - \b(내용)\ : 단어의 경계 (공백, 문장 부호 등)를 의미

- 수량 반복 지정 
    - \* : 앞 문자 0번 이상 반복 (없어도 됨)
    - \+ : 앞 문자 1번 이상 반복 (최소 하나는 있어야 함)
    - \? : 앞 문자 0번 또는 1번 (있거나 없거나)
    - \{n} : 정확히 n번 반복
    - {n,} : 최소 n번 이상 반복
    - {n,m} : n번 이상 m번 이하 반복

- 문자 집합 범위
    - . : 아무 문자나 딱 한 글자 (줄바꿈 문자 제외)
    - [ ]: 괄호 안의 문자 중 하나 (예: [aeiou] → 모음 중 하나)
    - [^ ]: 괄호 안의 문자를 제외한 나머지 (부정) 
    - \- : 범위를 지정 (예: [0-9] → 모든 숫자, [a-z] → 모든 소문자)
    - \| : OR(또는) 연산 (예: apple|banana → apple 혹은 banana)

- Escape : 메타문자를 일반문자로
    - \ : 메타문자 앞에 붙여서 문자 그대로 인식
