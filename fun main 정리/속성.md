- var : 변수
- val : 상수
- const val : 컴파일 전 확정되는 상수
    
    -> 위치 : object, companion object, 파일 최상위 (main 내 불가)
    - 형식
```kotlin
const val APP_NAME = "MyApp"

fun main() {
    println(APP_NAME)
}
```
---
- lateinit var : 나중에 초기화할 변수 
    
    -> var만 가능 
    - 형식
```kotlin
class User {
    lateinit var name: String

    fun initName() {
        name = "철수"
    }

    fun printName() {
        println(name)
    }
}

fun main() {
    val user = User()
    user.initName()
    user.printName()
} // 결과 : 출석
```
_초기화 확인_

```kotlin
class User {
    lateinit var name: String

    fun check() {
        println(this::name.isInitialized)
    }
}
```
---
- by lazy : 처음 사용 시 딱 1번 초가화 + __마지막 저장값이 재사용 시 출력__ + _변수명 뒤 작성_
    - 형식 
```kotlin
val message: String by lazy {
    println("초기화 실행")
    "안녕하세요"
}

fun main() {
    println("시작")
    println(message)
    println(message)
} /* 결과 : 시작
초기화 실행
안녕하세요
안녕하세요 */   
```
---
- vararg : __매개변수를 여러 개 받을 수 있게 함__ + _매개변수명 앞에 붙임_
    - 형식 
```kotlin
fun printNames(vararg names: String) {
    for (name in names) {
        println(name)
    }
}

fun main() {
    printNames("철수", "영희", "민수")
} /* 결과 : 철수
영희
민수 */
```
