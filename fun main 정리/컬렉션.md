- listOf(elements) :  읽기 전용(변경 X) 리스트 생성
    - 형식
```kotlin
fun main() {
    val list = listOf("A", "B", "C")
    println(list)
}
```

- mutableListOf(elements) : 수정 가능한 리스트
    - 형식
```kotlin
fun main() {
    val list = mutableListOf("A", "B")
    list.add("C")
    println(list)
}
```
---
- setOf(elements) : 중복 없는 읽기 전용 집합
    - 형식
```kotlin
fun main() {
    val set = setOf(1, 2, 2, 3)
    println(set)
} // [1, 2, 3]
```

- mutableSetOf(elements) : 수정 가능한 set
    - 형식
```kotlin
fun main() {
    val set = mutableSetOf("A", "B")
    set.add("B")
    set.add("C")
    println(set)
} // [A, B, C]
```
---
- mapOf(key to value) : 키-값 구조의 읽기 전용 맵 생성
    - 형식
```kotlin
fun main() {
    val map = mapOf("name" to "철수", "age" to 20)
    println(map)
} // {name=철수, age=20}
```
- mutableMapOf() : 수정 가능한 map 
    - 추가하는 법 : ___map[키] = 값___

    - 형식
```kotlin
fun main() {
    val map = mutableMapOf<String, Int>()
    map["apple"] = 3
    map["banana"] = 5
    println(map)
}
```
---
- arrayOf(elements) : 배열
    - 형식
```kotlin
fun main() {
    val arr = arrayOf(10, 20, 30)
    println(arr.joinToString())
}
```
- Array(size) { } : 람다로 배열 규칙

    - 형식 
```kotlin
fun main() {
    val arr = Array(5) { index -> index * 2 }
    println(arr.joinToString())
}
```
_2챠원형식_
```kotlin
fun main() {
    val arr = Array(3) { IntArray(3) }

    for (i in arr.indices) {
        println(arr[i].joinToString())
    }
} /* 0, 0, 0
    0, 0, 0
    0, 0, 0 */
```
